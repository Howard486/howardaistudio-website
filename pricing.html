常見問題 · FAQ

Frequently Asked Questions

Trial 結束後會自動扣款嗎？
Trial 不需信用卡即可使用。試用期結束後不會自動扣款，若要升級至 Pro 需至 Web App 內主動選擇方案並完成付款。
Pro 方案定價為何不顯示在網站？
我們有多個 Solution，定價依方案各有不同。為確保你看到最即時正確的資訊，Pro 方案定價統一在 Web App 內顯示，Sign up 後即可查看。
如何開始？需要 Global Admin 嗎？
不需要 Global Admin。僅需在你的 Entra ID 建立一個具備 Security Reader 角色的 App Registration。系統設定頁面提供詳細步驟引導，填入 Tenant ID、Client ID、Client Secret 後即可連線。LINE 推播另需設定 Channel Token 與 Target User ID。
可以隨時取消訂閱嗎？
是的，Pro 訂閱可隨時取消，不收違約金。取消後服務在當前計費週期結束時停止。詳細退款條件請參閱 退款政策。
M365 帳號自理方案何時推出？
目前正在積極開發中。如有興趣搶先體驗，歡迎 聯絡我們 加入 waitlist。
付款方式與發票
Pro 方案所有付款透過 Paddle 處理，支援 Visa、Mastercard、American Express 及 PayPal。如需台灣統編或年度合約請洽 Enterprise 方案。
